Contexte

PokeDeck est un site PHP, avec du contenu HTML et CSS. Le but de ce projet était de réaliser un Pokédex pour les utilisateurs leurs permettant d'ajouter ou de supprimer des pokémons à leur collection via des variables PHP. Dans un premier temps, nous avons dû réaliser un moodboard, gabarit, wireframes, etc… Une fois la partie graphique défini, nous avons commencer le HTML, CSS afin de finir par le PHP.

Logiciels utilisés

- Illustrator

- Photoshop

- Visual Studio Code

- Figma

Compétences utilisées

- HTML

- CSS

- PHP

- Graphisme

Experience acquises

Sur ce projet, les experiences acquises ont été de m'améliorer sur le CSS et HTML et de nettement progresser en PHP. Mais également d'apprendre les wireframes et d'utiliser Figma afin de créer une maquette de site.
